# 408考研刷题系统

## 项目介绍

408考研刷题系统是一个专为考研学生设计的在线刷题平台，支持数据结构、计算机组成原理、操作系统、计算机网络四个科目的题目练习和测试。

## 功能特点

### 学生端功能
- **用户登录/注册**：支持学生用户的注册和登录
- **科目选择**：选择要练习的科目
- **题库/历年真题**：查看题目、答题、提交答案、查看解析
- **错题本**：记录错题，支持重做和标记已掌握
- **成绩统计**：查看总体答题情况、科目统计、最近成绩趋势

### 技术栈
- **前端**：React 18 + Vite
- **后端**：Node.js + Express (待实现)
- **数据库**：MongoDB (待实现)

## 快速开始

### 前端运行

1. 进入前端目录
```bash
cd frontend-new
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm run dev
```

4. 访问 http://localhost:3001/
